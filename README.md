Morse Code 

== Overview ==

Morse code is a communication system that uses dots (.) and dashes (-) to represent letters, numbers, and symbols. Instead of sending full text, messages are encoded into signals that can be transmitted using sound, light, or electrical pulses.

It played an important role in the early development of long-distance communication.

== Background ==

Morse code was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail. It was created alongside the telegraph, which allowed messages to be sent over wires using electrical signals.

Before this, communication over long distances was slow and unreliable. Morse code made it possible to send messages almost instantly, which was a major breakthrough.

It was widely used in telegraph systems, maritime communication, and emergency signaling.

== How Morse Code Works ? ==

Each letter and number is represented by a unique combination of dots and dashes. A dot is a short signal, while a dash is a longer signal. These signals are sent in sequence to form words and sentences.

There are also pauses used in Morse code:

A short pause separates parts of the same letter
A medium pause separates letters
A longer pause separates words
Examples

Letters:
A = .-
B = -...

Words:
